About RPG Character Creator

This is a simple character creator for Dungeons and Dragons 5th Edition. It was Created by Jo Stephenson for the Mimo Bootcamp Unit 4 Project

It uses HTML, CSS and JavaScript to create a simple character creator that allows you to create a character and save it to local storage.

It uses the D&D 5th Edition API to pull class, race, spell and equipment information for your character

It also uses The Chartopia API to generate random names
